SEÇÃO 3: CÓDIGO-FONTE (Padrões em Sintaxe)
-----------------------------------------------------------------
public class ExemploCompressao {
    private static final int CONSTANTE = 100;

    public void metodoExemplo() {
        for (int i = 0; i < CONSTANTE; i++) {
            System.out.println("Iteração: " + i);
            processarDados(i);
        }
    }

    private void processarDados(int valor) {
        if (valor % 2 == 0) {
            System.out.println("Par: " + valor);
        } else {
            System.out.println("Ímpar: " + valor);
        }
    }
}

SEÇÃO 4: TEXTO NATURAL (Português)
-----------------------------------------------------------------
O algoritmo de Huffman é uma técnica de compressão sem perdas
desenvolvida por David A. Huffman em 1952. Ele utiliza uma estratégia
gulosa para construir uma árvore binária onde os caracteres mais
frequentes recebem códigos binários mais curtos, enquanto caracteres
menos frequentes recebem códigos mais longos.

A compressão funciona particularmente bem em textos com alta
redundância e padrões repetitivos. Por exemplo, em um texto em
português, as letras 'a', 'e', 'o' aparecem com muito mais
frequência que 'k', 'w', 'y'. O algoritmo explora essa propriedade
estatística da linguagem para reduzir o tamanho total dos dados.

SEÇÃO 3: CÓDIGO-FONTE (Padrões em Sintaxe)
-----------------------------------------------------------------
public class ExemploCompressao {
    private static final int CONSTANTE = 100;

    public void metodoExemplo() {
        for (int i = 0; i < CONSTANTE; i++) {
            System.out.println("Iteração: " + i);
            processarDados(i);
        }
    }

    private void processarDados(int valor) {
        if (valor % 2 == 0) {
            System.out.println("Par: " + valor);
        } else {
            System.out.println("Ímpar: " + valor);
        }
    }
}

SEÇÃO 4: TEXTO NATURAL (Português)
-----------------------------------------------------------------
O algoritmo de Huffman é uma técnica de compressão sem perdas
desenvolvida por David A. Huffman em 1952. Ele utiliza uma estratégia
gulosa para construir uma árvore binária onde os caracteres mais
frequentes recebem códigos binários mais curtos, enquanto caracteres
menos frequentes recebem códigos mais longos.

A compressão funciona particularmente bem em textos com alta
redundância e padrões repetitivos. Por exemplo, em um texto em
português, as letras 'a', 'e', 'o' aparecem com muito mais
frequência que 'k', 'w', 'y'. O algoritmo explora essa propriedade
estatística da linguagem para reduzir o tamanho total dos dados.
